# Intelligent Agriculture System

**Team 38 project** for the course **ICT Project Management**.

A web-based intelligent agriculture system for collecting agricultural data, analyzing it, and generating AI-based recommendations for irrigation, crop protection, activity planning, and yield improvement.

---

## Project Overview

The **Intelligent Agriculture System** is designed to help users manage and analyze agricultural information in a structured and efficient way. The system will allow users to enter and manage crop-related data, store information in a database, use external APIs for additional data such as weather conditions, and apply artificial intelligence to generate useful recommendations.

The main goal of the project is to support better decision-making in agriculture through a digital platform that combines:
- database management,
- file import/export,
- external API integration,
- AI-based recommendation generation.

## Main Features

- User registration and login
- Agricultural data entry and management
- Crop, parcel, and activity tracking
- Database storage and retrieval
- External API integration for relevant agricultural data
- AI-generated recommendations
- Import and export of data files
- Admin management functionalities

---

## Planned System Modules

- **Frontend** – user interface for interaction with the system
- **Backend** – business logic, REST API, and system services
- **Database** – structured storage of all application data
- **AI Module** – recommendation generation based on available input data
- **External API Integration** – weather or agricultural data services
- **Documentation** – project specifications, reports, diagrams, and notes

## Planned Technology Stack

The exact implementation may evolve during development, but the currently planned stack includes:
- **Frontend:** React
- **Backend:** Spring Boot
- **Database:** PostgreSQL
- **Version Control:** Git
- **Project Management:** Jira
- **AI Integration:** AI-based recommendation service
- **External APIs:** Weather or agriculture-related API
